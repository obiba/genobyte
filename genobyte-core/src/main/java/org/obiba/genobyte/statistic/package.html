Provides a way to calculate statistics on the store's set of data. The module consists essentially of these parts:
  • Statistic implementations, which knows how to calculate a given statistic.
  • StatsRunDefinition, which defines batches of statistics to be computed at a given time.
  • StatsPool, which calculates statistics in the correct order and keeps the results.
  • StatsDigester, which knows what to do with pool data after statistics have been calculated.
